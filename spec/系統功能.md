## 二、前台功能（無需登入）

### 2.1 揪作列表頁（`/`）

- 顯示所有揪作，按建立時間降序排列
- 分類篩選 tab：全部 / 活動策劃 / 協作計畫 / 內容媒體 / 空間場域 / 研究調查
- 每頁 12 筆，超過顯示分頁
- 卡片固定尺寸（寬 288px × 高 288px），以 flex-wrap 排列
- 卡片顯示：分類 badge、發起日期（格式：發起：2026年6月9日）、標題、描述、發起人名稱、留言數、加入人數
- 右上角：深色/淺色模式切換（🌙/☀️）、管理員登入（🔑）、發起揪作按鈕

### 2.2 揪作詳情頁（`/wish/[id]`）

- 顯示：分類 badge、發起日期（格式：2026年6月9日）、完整標題
- 完整描述內容
- 發起人名稱及社群帳號（IG / Threads）
- 已加入人員列表（名稱、社群帳號、簡短說明）
- 「我想加入！」功能（展開表單，填寫名稱、社群帳號選填、自我介紹選填）
- 留言區：預設顯示最新 3 則，超過可展開全部 / 收起
- 留言表單：填寫名稱（必填）、社群帳號（選填）、留言內容（必填）

### 2.3 發起揪作頁（`/wish/new`）

- 填寫欄位：
  - 主題標題（必填，最多 50 字）
  - 分類（必填）
  - 描述（必填，最多 500 字）
  - 發起人名稱（必填，最多 20 字）
  - IG / Threads 帳號（選填，需以 @ 開頭）
- 送出後直接公開，無需審核
- 含 honeypot 隱藏欄位防垃圾投稿

## 三、管理後台（需登入）

### 3.1 登入頁（`/admin`）

- Email + 密碼登入
- 帳號資料儲存於 Neon admin_accounts 表
- 登入失敗顯示錯誤提示

### 3.2 後台 Dashboard（`/admin/dashboard`）

四個功能 Tab：

#### Tab 1：許願

- 列出所有揪作
- 新增揪作（填寫標題、分類、描述、發起人、社群帳號）
- 編輯現有揪作（所有欄位皆可修改）
- 刪除揪作（連同留言和加入記錄一併刪除）

#### Tab 2：留言

- 列出所有留言，顯示所屬揪作標題與留言者
- 編輯留言內容
- 刪除留言

#### Tab 3：分類

- 列出所有分類（slug + 顯示名稱）
- 新增分類（填寫 slug 和顯示名稱）
- 編輯分類（修改 slug 和顯示名稱）
- 刪除分類（注意：現有揪作不會自動更新）

#### Tab 4：帳號

- 列出所有管理員帳號
- 新增管理員帳號（Email + 密碼，至少 8 個字元）
- 編輯帳號（修改 Email 或密碼）
- 刪除帳號（保護機制：不能刪除自己、至少保留一個帳號）

## 四、資料結構

### Wishes（揪作）

| 欄位 | 型別 | 說明 |
|------|------|------|
| id | uuid | 主鍵 |
| title | text | 標題（最多 50 字） |
| category | text | 分類 slug |
| description | text | 描述（最多 500 字） |
| author_name | text | 發起人名稱 |
| author_social | text | 社群帳號（選填） |
| created_at | timestamptz | 建立時間 |

### Comments（留言）

| 欄位 | 型別 | 說明 |
|------|------|------|
| id | uuid | 主鍵 |
| wish_id | uuid | 所屬揪作（cascade delete） |
| author_name | text | 留言者名稱 |
| author_social | text | 社群帳號（選填） |
| content | text | 留言內容（最多 300 字） |
| created_at | timestamptz | 建立時間 |

### Joins（加入意願）

| 欄位 | 型別 | 說明 |
|------|------|------|
| id | uuid | 主鍵 |
| wish_id | uuid | 所屬揪作（cascade delete） |
| name | text | 名稱 |
| social | text | 社群帳號（選填） |
| message | text | 自我介紹（選填，最多 100 字） |
| created_at | timestamptz | 建立時間 |

### Categories（分類）

| 欄位 | 型別 | 說明 |
|------|------|------|
| id | uuid | 主鍵 |
| slug | text | 英文代號（唯一） |
| name | text | 顯示名稱 |
| created_at | timestamptz | 建立時間 |

### Admin Accounts（管理員帳號）

| 欄位 | 型別 | 說明 |
|------|------|------|
| id | uuid | 主鍵 |
| email | text | Email（唯一） |
| password_hash | text | bcrypt 雜湊密碼 |
| created_at | timestamptz | 建立時間 |

## 五、API 路由

### 公開 API

| 方法 | 路徑 | 說明 |
|------|------|------|
| GET | /api/wishes | 揪作列表（支援 ?category= / ?page=） |
| POST | /api/wishes | 新增揪作 |
| GET | /api/wishes/[id] | 揪作詳情（含留言、加入人） |
| POST | /api/wishes/[id]/comments | 新增留言 |
| POST | /api/wishes/[id]/join | 提交加入意願 |

### 管理員 API（需登入 session）

| 方法 | 路徑 | 說明 |
|------|------|------|
| POST | /api/admin/wishes | 新增揪作（後台） |
| PATCH | /api/admin/wishes/[id] | 編輯揪作 |
| DELETE | /api/admin/wishes/[id] | 刪除揪作 |
| PATCH | /api/admin/wishes/[id]/urgent | 切換急需標記 |
| GET | /api/admin/comments | 所有留言列表 |
| PATCH | /api/admin/comments/[id] | 編輯留言 |
| DELETE | /api/admin/comments/[id] | 刪除留言 |
| GET | /api/admin/categories | 分類列表 |
| POST | /api/admin/categories | 新增分類 |
| PATCH | /api/admin/categories/[id] | 編輯分類 |
| DELETE | /api/admin/categories/[id] | 刪除分類 |
| GET | /api/admin/account | 管理員帳號列表 |
| POST | /api/admin/account | 新增管理員帳號 |
| PATCH | /api/admin/account/[id] | 編輯管理員帳號 |
| DELETE | /api/admin/account/[id] | 刪除管理員帳號 |
